Este repositório contém os exercícios da disciplina de Linguagem de Programação 1 (LP1) do curso de Tecnologia da Informação (BTI) da UFRN. Cada pasta ou arquivo corresponde a uma atividade prática realizada em aula para fixação dos conceitos aprendidos.


This repository contains exercises from the Programming Language 1 (LP1) course, part of the Information Technology (BTI) degree at UFRN. Each folder or file represents a practical exercise conducted during classes to reinforce the learned concepts.
